## 📌 Overview

A fully responsive personal portfolio built with vanilla HTML, CSS, and JavaScript — featuring a **Midnight Terminal** aesthetic with amber gold and electric cyan accents. Includes smooth scroll animations, a typing hero, dark/light toggle, and a working contact form via EmailJS.

**Live at → [sridevbaag.github.io](https://sridevbaag.github.io/)**

---

## ✨ Features

- ⚡ **Preloader** with animated `{SB}` logo
- 🎯 **Typing hero** — cycles through what I build
- 🌙 **Dark / Light theme** toggle with localStorage persistence
- 📜 **Scroll progress bar** at the top of the page
- 🖱️ **Custom animated cursor** with follower ring
- 🎞️ **Scroll-reveal animations** on every section
- 📊 **Animated stat counters** in the About section
- 💼 **Experience timeline** — internship + education
- 📬 **Contact form** powered by EmailJS with status feedback
- 📱 **Fully mobile responsive** with hamburger menu

## 🚀 Run Locally

No build step needed — it's pure HTML/CSS/JS.

```bash
# Clone the repo
git clone https://github.com/Sridevbaag/Sridevbaag.github.io.git

# Open in browser
cd Sridevbaag.github.io
open index.html
# or just drag index.html into Chrome
```

> To enable the contact form locally, replace the EmailJS public key, service ID, and template ID in `script.js` with your own credentials from [emailjs.com](https://emailjs.com).
